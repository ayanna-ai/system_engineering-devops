 # 0x02. Shell, I/O Redirections and Filters
This directory contains shell script files for learning about input/output redirections, pipes, filters, and text processing in Linux.
## Learning Objectives
- Understanding standard input, output, and error
   - Using redirections (>, >>, <, |)
   - Working with filters (grep, sort, uniq, etc.)
   - Combining commands with pipes
   - Text processing and manipulation

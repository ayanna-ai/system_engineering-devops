# 0x04. Loops, conditions and parsing 
   This directory contains Bash scripts for learning about loops, conditional statements, and text parsing.
   ## Learning Objectives
   - Understanding loops: for, while, until
   - Conditional statements: if, else, elif, case
   - Comparison operators
   - File test operators
   - Parsing and text processing
   - Using `cut`, `read`, and other parsing commands
   - Script automation and repetitive tasks

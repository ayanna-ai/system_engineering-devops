# 0x03. Shell, init files, variables and expansions 
   This directory contains shell script files for learning about shell initialization files, variables, expansions, and aliases.
   ## Learning Objectives
   - Understanding shell initialization
   - Working with local and environment variables
   - Creating and using aliases
   - Understanding expansions (brace, tilde, parameter, command substitution)
   - Arithmetic operations in shell
   - The difference between local and environment variables
